# 📊 PhonePe Transaction Analysis Dashboard

## Project Overview
This project analyzes over 100,000 PhonePe transaction records using Microsoft Excel and Power BI. The dashboard provides insights into transaction trends, state-wise performance, user activity, and payment behavior across different categories.

## Objectives
- Analyze transaction volume and value trends
- Compare state-wise transaction performance
- Identify user growth patterns
- Visualize payment category distribution
- Create an interactive dashboard for business insights

## Tools Used
- Microsoft Excel
- Power BI

## Dashboard Pages
1. Executive Overview
2. Transaction Analysis
3. State-wise Performance
4. User Analysis
5. Trend Analysis

## Key Insights
- Transaction growth trends over time
- State-wise transaction performance
- User activity analysis
- Category-wise transaction distribution
- Regional comparison of transaction volume
